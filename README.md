Vermeer Installaties — Loodgieter Website

Live site: vermeer-installaties.netlify.app

A modern, responsive one-page website for a fictional local plumbing company in Twente, the Netherlands. Built as a portfolio project to demonstrate web development skills.

📋 Project Overview
TypeFictief portfolio projectNicheLokaal installatiebedrijfStackHTML5, CSS3, Vanilla JavaScriptDesignOne-pager, mobile-first, responsive

🛠️ Features

Sticky navbar met hamburger menu op mobiel
Animated hero sectie met statistieken
USP balk
Diensten grid met featured card
Over ons sectie met statistieken
Werkgebied overzicht (Twente)
Offerte aanvraagformulier met success state
Contact sectie
Scroll animaties via Intersection Observer
Volledig responsive (mobiel, tablet, desktop)


🧰 Tech Stack

HTML5 (semantische structuur)
CSS3 (custom properties, grid, flexbox, animaties)
Vanilla JavaScript (DOM manipulatie, scroll animaties, form handling)
Google Fonts (Bebas Neue + DM Sans)
